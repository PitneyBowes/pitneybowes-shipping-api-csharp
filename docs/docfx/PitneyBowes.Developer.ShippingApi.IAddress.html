AddressLines

Street address and/or apartment and/or P.O. Box. You can specify up to three address lines. For USPS domestic destinations, ensure that the street address is specified as the last of the 3 address lines.This way, the street address is printed right above the city, state, postal zip code, per USPS label guidelines.

See for considerations when specifying multiple lines in a shipment's fromAddress when MINIMAL_ADDRESS_VALIDATION is enabled.
